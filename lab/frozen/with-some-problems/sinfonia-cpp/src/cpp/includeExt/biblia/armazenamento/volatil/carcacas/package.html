Visão Geral do Pacote Carcaças

Contém carcaças para o armazenamento de informações. Essas carcaças são estruturas de dados para prover acesso aos dados armazenados. É possível armazenar as mesmas informações de diversas maneiras diferentes, e a melhor maneira está diretamente ligada ao tipo do acesso à informação: se o acesso mais adequado for através de índice inteiro por exemplo, uma Lista seria uma boa saída. Se esse índice puder ser qualquer tipo de dado, o Mapa é mais adequado. Para armazenamento de Informação (biblia::armazenamento::volatil::Informacao) de forma estruturada, use um Composto. Se os dados não têm tipo use um DadosSemTipo.

As carcaças base deste pacote são: Lista, Mapa, DadosSemTipo e Composto.
