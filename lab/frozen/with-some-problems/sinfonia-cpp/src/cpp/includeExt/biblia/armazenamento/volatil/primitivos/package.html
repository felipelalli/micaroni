Visão Geral do Pacote Primitivos

Os tipos primitivos previstos neste pacote encapsulam os tipos primitivos da linguagem. Isso é útil para manutenções futuras e para centralizar um grupo específico de tipos em uma classe conhecedora desses tipos, que no caso é a classe ConhecedorDePrimitivos.

Os tipos primitivos previstos neste pacote são: Boleano, Flutuante, Inteiro, Relativo e String.
