1. Definição detalhada do nível 0

1.1. Nível 0a

Consiste basicamente numa cadeia de caracteres (string), definida pelo padrão UTF-8.

1.1.1. Interface

(nivel0a? ‹obj›)

Verifica se o ‹obj› é do tipo nivel0a. Se a entrada for uma string qualquer, o resultado deve ser #f.

(string->nivel0a ‹string›)
(nivel0a->string ‹obj-nivel0a›)

Converte uma string num objeto de nível 0a e vice-versa.

  • ‹string› é uma string qualquer, como por exemplo "" ou "isto é uma frase".
  • ‹obj-nivel0a› é um objeto que satisfaz a condição (nivel0a? ‹obj-nivel0a›).
Se o tipo recebido no parâmetro não for o esperado, as funções devem retornar #f.

Teste:

  • (nivel0a->string (string->nivel0a "esta frase")) ⇒ "esta frase"
  • (string->nivel0a 10) ⇒ #f

1.2. Nível 0b

Consiste basicamente numa string UTF-8 com os símbolos do Alfabeto fonético internacional (AFI ou IPA). Ver tabela em Unicode to CXS Mapping para saber o código Unicode do símbolo.

1.2.1. Interface

(simbolo-IPA? ‹caractere›)

Verifica se o caractere passado como parâmetro é algum símbolo válido do IPA.

(string-IPA? ‹string›)

Verifica se cada caractere da string passada como parâmetro corresponde à condição de (simbolo-IPA? ‹caractere›). Uma string vazia retorna #t.

(nivel0b? ‹obj›)

Verifica se o ‹obj› é do tipo nivel0b. Se a entrada for uma string qualquer, o resultado deve ser #f.

(string->nivel0b ‹string›)
(nivel0b->string ‹obj-nivel0b›)

Converte uma string num objeto de nível 0b e vice-versa.

  • ‹string› deve ser uma string que satisfaz a condição (string-IPA? ‹string›).
  • ‹obj-nivel0b› é um objeto que satisfaz a condição (nivel0b? ‹obj-nivel0b›).
Se o tipo recebido no parâmetro não for o esperado, as funções devem retornar #f.

1.3. Nível 0 (0a + 0b)

O nível 0 consiste num par contendo um objeto do nível 0a e um objeto do nível 0b. É possível que um objeto do nível 0 possua apenas o nível 0a ou apenas o nível 0b. Pode ser também que um objeto do nível 0 não possua nenhum subnível associado, sendo considerado então um objeto "vazio".

1.3.1. Interface

(nivel0? ‹obj›)

Verifica se o ‹obj› é do tipo nivel0. Se a entrada for um par, o resultado deve ser #f. Se a entrada for algum objeto do tipo nivel0a ou nivel0b, o resultado também deve ser #f.

(cria-nivel0 ‹obj-nivel0a› ‹obj-nivel0b›)

Cria e retorna um objeto no nível 0 com base em um do nível 0a e outro do nível 0b. Se o parâmetro não corresponder com o esperado a função retorna #f. O ‹obj-nivel0a› deve ser um objeto do nível 0a ou #f caso ele esteja ausente. O ‹obj-nivel0b› deve ser um objeto do nível 0b ou #f caso ele esteja ausente. A função pode ser invocada desta forma: (cria-nivel0 #f #f) caso o nível 0 não possua nenhum subnível.

(contem-nivel0a? ‹obj-nivel0›)
(contem-nivel0b? ‹obj-nivel0›)

Retorna #f caso o nível 0 passado como parâmetro não contenha nenhum objeto do nível 0a (ou 0b) associado e #t caso contrário. Também retorna #f caso o objeto passado como parâmetro não for do tipo esperado.

(acessa-nivel0a ‹obj-nivel0›)
(acessa-nivel0b ‹obj-nivel0›)

Retorna o objeto de nível 0a (ou nível 0b) que o objeto de nível 0 passado como parâmetro contém. Retorna #f caso não contenha objeto do nível 0a (ou nível 0b) ou caso o objeto passado como parâmetro não seja do tipo nível 0.

(define-nivel0a! ‹obj-nivel0› ‹obj-nivel0a›)
(define-nivel0b! ‹obj-nivel0› ‹obj-nivel0b›)

Redefine o objeto de nível 0a (ou nível 0b) que o objeto de nível 0 passado como parâmetro contém. Retorna #f caso os objetos passados como parâmetro não sejam dos tipos esperados ou #t se foi redefinido com sucesso.
